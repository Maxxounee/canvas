# Shortcuts

### Work with files and IDE

`Cmd Shift N` - Find in files  
`Cmd Shift A` - Find in actions  
`Cmd Shift M` - New scratch file  
`Cmd Shift B` - Terminal  
`Cmd 1` - Project files tab  
`Cmd E` - Recent Files  
`Cmd 0` - Commit tab

### Work with code

`Cmd W` - Extend selection  
`Cmd Shift ↓↑` - Move expansion ↓↑  
`Cmd D` - Duplicate
